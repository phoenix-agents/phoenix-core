🤖 新建 Bot

📋 准备步骤
  1. 在 Discord 开发者门户 创建新 Bot
  2. 复制 Bot Token（点击 "Reset Bot Token"）
  3. 复制 Client ID（在 "OAuth2" 或应用信息页面）
  4. 复制 Server ID（在 Discord 中右键点击服务器 > 复制 ID）
  5. 复制 Channel ID（在 Discord 中右键点击频道 > 复制 ID）
  6. 在此填写配置并保存
💡 提示：需要先在 Discord 用户设置 > 高级 中开启 "开发者模式" 才能复制 ID
Token 只会保存在本地 .env 文件中，不会上传
