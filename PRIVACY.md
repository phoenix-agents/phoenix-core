# 隐私政策

**最后更新**: 2026-04-12  
**生效日期**: 2026-04-12

## 📌 概述

Phoenix Core 尊重并保护用户隐私。本政策说明我们如何处理用户数据。

## 核心原则

1. **数据本地存储** - 所有用户数据（记忆、技能、配置）存储在本地
2. **零收集默认** - 默认不收集任何用户数据
3. **用户完全控制** - 用户可以随时查看、修改、删除自己的数据
4. **透明度高** - 所有数据处理逻辑公开可查

## 📊 数据类型

### 本地数据（用户所有）

| 数据类型 | 存储位置 | 用途 |
|---------|---------|------|
| 记忆文件 | `./memories/` | 会话记忆、用户偏好 |
| 技能文件 | `./skills/` | 技能定义、执行记录 |
| 配置文件 | `./config.json`, `./.env` | 系统配置 |
| 会话数据 | `./main.db` | 对话历史（SQLite） |
| 日志文件 | `./logs/` | 运行日志 |

### 可选遥测数据（需用户同意）

如果用户选择加入匿名改进计划（`PHOENIX_TELEMETRY_OPT_IN=true`），可能收集：

| 数据类型 | 处理方式 | 用途 |
|---------|---------|------|
| 功能使用频率 | 脱敏聚合 | 改进产品 |
| 技能类型分布 | 哈希处理 | 了解使用模式 |
| 成功率统计 | 分桶聚合 | 优化性能 |

**绝不收集**:
- ❌ API Key
- ❌ 业务数据
- ❌ 技能内容
- ❌ 用户个人信息
- ❌ 原始对话内容

## 🔐 数据安全

### 技术措施
- 敏感信息（API Key）存储在 `.env` 文件，建议设置文件权限为 600
- `.gitignore` 默认排除敏感配置文件
- 数据库使用 SQLite，支持文件级加密（可选）

### 建议实践
```bash
# 设置文件权限
chmod 600 .env
chmod 600 config.json
chmod 700 workspaces/
```

## 👤 用户权利

根据 GDPR/CCPA 等隐私法规，用户拥有：

1. **访问权** - 查看存储的所有数据
2. **更正权** - 修改不准确的数据
3. **删除权** - 要求删除个人数据
4. **导出权** - 导出所有数据
5. **反对权** - 反对数据处理

行使方式：
```bash
# 查看所有记忆
cat memories/MEMORY.md
cat memories/USER.md

# 导出所有数据
tar -czf phoenix-backup.tar.gz memories/ skills/ config.json

# 删除所有数据
rm -rf memories/ skills/ main.db
```
