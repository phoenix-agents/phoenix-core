# 网络学习技能使用指南

## 功能说明

这个技能让 Bot 能够：
1. **从网址学习** - 抓取网页内容并提取知识
2. **从搜索学习** - 搜索网络并整合多个来源
3. **自动转化为技能** - AI 分析后生成可复用的技能

## 使用方法

### 方法 1: 从 URL 学习

**Discord 命令示例**：
```
@运营 学习这个网址：https://example.com/live-streaming-tips
主题是：直播运营技巧
```

**Python 代码示例**：
```python
from skills.web_learning_skill import WebLearningSkill

skill = WebLearningSkill(bot_name="运营")
result = skill.learn_from_url(
    url="https://example.com/article",
    topic="直播运营技巧"
)

print(f"学习了 {result['knowledge_points']} 个知识点")
print(f"是否生成技能：{result['skill_created']}")
```

### 方法 2: 从搜索学习

**Discord 命令示例**：
```
@运营 从网上学习"抖音直播算法 2026"
```

**Python 代码示例**：
```python
skill = WebLearningSkill(bot_name="运营")
result = skill.learn_from_search(
    query="抖音直播算法 2026",
    num_results=5  # 分析前 5 个搜索结果
)
```

## API 配置

需要设置环境变量：

```bash
export DASHSCOPE_API_KEY="your-api-key-here"
```

## 学习记录

所有学习记录会保存到：
- `workspaces/<bot>/DYNAMIC/learnings/YYYY-MM-DD.md` - 学习日志
- `workspaces/<bot>/memory/知识库.md` - 结构化知识
- `workspaces/<bot>/DYNAMIC/skills/` - 可复用技能

## 查看已学习的知识

**Discord 命令**：
```
@运营 展示知识库
```

**查看技能列表**：
```bash
ls workspaces/运营/DYNAMIC/skills/
```

## 最佳实践

1. **选择高质量来源** - 优先学习知乎、公众号、专业博客
2. **指定明确主题** - 告诉 Bot 学习的具体方向
3. **定期整理** - 每月回顾知识库，删除过时内容
4. **技能复用** - 检查已有技能，避免重复
